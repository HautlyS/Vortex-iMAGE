# Technology Stack

This document outlines the technology stack used in the project.

## Backend

- **Language:** Rust
- **Framework:** Tauri (v2)

## Frontend

- **Language:** TypeScript
- **Framework:** Vue.js (v3)
- **Bundler:** Vite
- **Styling:** Tailwind CSS (v4)

## Authentication

- **Method:** GitHub OAuth Device Flow

## Platform

- **Desktop:** Windows, macOS, Linux
- **Mobile:** Android, iOS
