# Product Guidelines

This document outlines the guidelines for the product's prose style, brand messaging, and visual identity.

## Prose Style

All user-facing content should be **technical and precise**. The language should be accurate, detailed, and suitable for a developer audience.

## Brand Messaging

The core brand message is: **"Your photos, your code, your control."** This highlights developer ownership and privacy.

## Visual Identity

The application should have a **modern and vibrant** look and feel. This includes using bold colors and animations to create a dynamic and engaging user experience.
